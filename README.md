# BDP-2020 (archive)

Mobile application capable of blurring vehicle license plates. <b>Used as an integration practice between flutter and tensorflow lite.</b> Made for personal use in two days of development. The first day was used for data extraction/machine learning model development and the second day for mobile development. It has a simple but functional interface.

## Data extraction

Data were extracted from internet pages, after which they were manually labeled using the [labelme][labelme-link] tool.

## Machine Learning Model

Initially, a model of marking boxes with perspective was used. However, after testing, it was decided to use an unet-style segmentation model, due to the ease of mobile implementation and the result being satisfactory.

The code was written using the keras framework, using Mobilenet-v2 as encoder.

After that the model was converted to tflite using the following code:

```python
converter = tf.lite.TFLiteConverter.from_keras_model(model)
tflite_model = converter.convert()
open("plate_blur256.tflite", "wb").write(tflite_model)
```

## Flutter App

Extremely simple application, it only has a grid with already processed images and a button to select an image from the gallery or take a photo with the camera. Nothing special was used for state management due to simplicity. The files are saved in a special folder and appear in the gallery.

## Flutter Tflite Integration

Initially the [script][script-link] was added and executed, to add the necessary dependencies.

In sequence created the classifier class, responsible for resizing the input to 256x256 and mapping the output to the place where the blur processing should be added.

[labelme-link]: https://github.com/wkentaro/labelme
[script-link]: https://github.com/am15h/tflite_flutter_plugin/blob/master/install.sh
